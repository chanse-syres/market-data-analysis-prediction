# market-data-analysis

Idea/Goal: A data analysis application for the stock market that uses live data, Data Science Analysis with ML methods, and AI along with the most up to date technology to predict the probability of a daily positive return. 

The team includes:
  - Data Analysist
  - Machine Learning Expert
  - Financial Strategist
  - Data Quality Professional
  - Cybersecurity Engineer
  - Project Supervisor

This is an experimental project that uses paper-money and should not be taken as serious financial advice.
